# isrl-research.github.io

Source for the [iSRL](https://isrl-research.github.io) website — 
the Interdisciplinary Systems Research Lab's public-facing research commons.

Built with [Quarto](https://quarto.org). Deployed via GitHub Actions to GitHub Pages.

## Structure

- `pub/` — published reports and datasets, each in its own subdirectory
- `team/` — team and contributor pages
- `assets/` — brand assets, stylesheets
- `_includes/` — shared Quarto partials
- `.github/workflows/` — publish pipeline

## Contributing

This repository is the website source, not the research repository. 
Research discussions, data, and sprint work live at 
[isrl-research/sandbox-research](https://github.com/isrl-research/sandbox-research).
